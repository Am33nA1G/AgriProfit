       Logic flow:
       a. Call compare_mandis() with TransportCompareRequest(commodity=commodity, quantity_kg=100.0, source_district=district, source_state="Unknown", limit=50, max_distance_km=max_distance_km) and db.
       b. Determine data_reference_date: scan all MandiComparison.latest_price_date values, take the MAX. If none, query DB: SELECT MAX(price_date) FROM price_history JOIN commodities WHERE commodities.name ILIKE commodity.
       c. Compute freshness cutoff: data_reference_date - timedelta(days=7).
       d. For each MandiComparison, compute:
          - days_since_update = (data_reference_date - comp.latest_price_date).days if latest_price_date else 999
          - is_stale = days_since_update > 7
          - stale_warning = f"Data last updated {comp.latest_price_date} — signal may be outdated" if is_stale else None
       e. Apply margin threshold: margin_pct = (comp.net_profit / comp.gross_revenue) * 100 if comp.gross_revenue > 0 else 0. If margin_pct < settings.arbitrage_margin_threshold_pct: increment suppressed_count, skip this result.
       f. Convert passing MandiComparison items to ArbitrageResult:
          - freight_cost_per_quintal = round(comp.costs.total_cost / 100.0 * 100, 2)  # costs.total_cost is for 100kg (1 quintal), so direct value
          - net_profit_per_quintal = round(comp.profit_per_kg * 100, 2)
          - price_date = comp.latest_price_date (use data_reference_date as fallback if None)
       g. Sort by net_profit_per_quintal descending, take top 3.
       h. Return ArbitrageResponse with has_stale_data = any(r.is_stale for r in results).

       CRITICAL: freight_cost_per_quintal calculation note — compare_mandis() is called with quantity_kg=100 (1 quintal). costs.total_cost from CostBreakdown is the TOTAL cost for that 100kg shipment. So freight_cost_per_quintal = costs.total_cost directly (already per-quintal). Verify this interpretation and document it clearly in code comments.
