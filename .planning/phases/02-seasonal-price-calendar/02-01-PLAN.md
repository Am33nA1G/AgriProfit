Create `backend/app/ml/seasonal/aggregator.py` with:

1. `load_and_prepare(parquet_path: str | Path, engine) -> pd.DataFrame`
   - Reads parquet with columns=["commodity", "state", "price_modal", "date"] using pyarrow engine
   - Drops rows where price_modal is null
   - Parses date column to datetime, extracts month (int 1-12) and year (int)
   - Joins price_bounds from DB: `pd.read_sql("SELECT commodity, lower_cap, upper_cap FROM price_bounds", con=engine)`
   - Clips price_modal: `df["price_modal"] = df["price_modal"].clip(lower=df["lower_cap"], upper=df["upper_cap"])`
   - Returns df with columns: commodity, state, price_modal, month, year (drops lower_cap, upper_cap)
   - NOTE: load_and_prepare DOES have DB access (reads price_bounds) — only compute_seasonal_stats must be pure

2. `compute_seasonal_stats(df: pd.DataFrame) -> pd.DataFrame`
   - Input df has columns: commodity, state, price_modal, month, year
   - PURE function — no DB calls, no file I/O
   - Compute years_of_data: `df.groupby(["commodity","state"])["year"].nunique()` (standard groupby is fine for nunique)
   - Compute per-month aggregates using explicit for-loop over `df.groupby(["commodity","state","month"])` — builds list of dicts, then pd.DataFrame(rows). This is the pandas 2.x safe pattern from STATE.md. Compute: median_price, q1_price (quantile 0.25), q3_price (quantile 0.75), iqr_price, record_count.
   - Merge years_of_data into agg df on (commodity_name, state_name)
   - Rank months: explicit for-loop over `agg.groupby(["commodity_name","state_name"])`, sort by median_price descending, assign month_rank=range(1,N+1). If group years_of_data >= 3: is_best = (month_rank <= 2), is_worst = (month_rank == len_group). Else: is_best=False, is_worst=False.
   - Return pd.concat(ranked_rows, ignore_index=True)

3. `upsert_seasonal_stats(stats_df: pd.DataFrame, engine) -> int`
   - Upserts all rows into seasonal_price_stats using ON CONFLICT (commodity_name, state_name, month) DO UPDATE SET
   - Uses SessionLocal() (same pattern as clean_prices.py upsert_price_bounds)
   - Returns count of rows written
