Implement `is_deficient(profile: dict, nutrient: str) -> bool`:
- Returns True when `profile.get(nutrient, {}).get("low", 0) > DEFICIENCY_THRESHOLD`

Implement `score_crop(profile: dict, tolerances: dict) -> float`:
- The function receives a single crop_row dict with keys: crop_name, nutrient, min_tolerance, ph_min, ph_max
- For the nutrient in that row, check if the block profile's category distribution is acceptable for this crop's min_tolerance
- Scoring logic: a crop with min_tolerance="low" is acceptable even when block has high low_pct (it can handle poor soil)
  - tolerance_level = LEVEL_ORDER[tolerances["min_tolerance"]]  (0=low, 1=medium, 2=high)
  - block_low_pct = profile.get(tolerances["nutrient"], {}).get("low", 0)
  - block_medium_pct = profile.get(tolerances["nutrient"], {}).get("medium", 0)
  - If tolerance_level == 0 (low): score = (block_low_pct + block_medium_pct) / 100.0 + 1.0
  - If tolerance_level == 1 (medium): score = block_medium_pct / 100.0 + 0.5 if not is_deficient(profile, tolerances["nutrient"]) else 0.0
  - If tolerance_level == 2 (high): score = 0.0 if is_deficient(profile, tolerances["nutrient"]) else block_medium_pct / 100.0
  - Return score

Implement `rank_crops(profile: dict, crop_rows: list) -> list`:
- For each crop_row dict in crop_rows, call score_crop(profile, crop_row)
